# Journal TUI - Python

### Introduction
A TUI Journal program written in Python.

### Installation

```bash
git clone https://github.com/harshdeepsinghin/Journal-TUI.git
cd Journal-TUI/Python
```

Setup a virtual environment.

```bash
python3 -m venv .venv
source .venv/bin/activate
```

Run the program

```bash
python3 Journal.py
```
